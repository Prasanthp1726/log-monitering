# E-Commerce Demo (Login page)

This is a minimal React + Vite scaffold demonstrating a responsive login page using Bootstrap.

Quick start:

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

Demo credentials: `user@example.com` / `password`

Features implemented:
- Product listing (fetched from https://fakestoreapi.com)
- Add to cart / Remove from cart / Decrement quantity / Clear cart
- Product search and category filter
- Responsive UI using Bootstrap
- State management with Redux Toolkit
- Routing with React Router
- API integration using Axios

Next steps: wire authentication, persist cart to localStorage, or style UI further.
